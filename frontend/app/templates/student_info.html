Proyecto 3 - Sistema de Facturación de Infraestructura en la Nube

Documentación del Proyecto

Descripción

Sistema de facturación para servicios de infraestructura en la nube desarrollado para Tecnologías Chapinas S.A. El sistema permite gestionar recursos, categorías, configuraciones, clientes e instancias, así como el registro y facturación de consumos.

Tecnologías Utilizadas

Backend

Flask 2.2.5 + Python

Frontend

Django 4.2.10 + Tailwind CSS

Base de Datos

XML Persistente

Comunicación

REST API (HTTP/JSON)

Funcionalidades Principales

  • Carga de configuraciones mediante XML (recursos, categorías, clientes, instancias)
  • Registro de consumos de recursos
  • Facturación automática por rango de fechas
  • Gestión CRUD de todos los recursos
  • Consulta de datos del sistema
  • Inicialización/limpieza del sistema
  • Validación de NITs y fechas con expresiones regulares
  • Almacenamiento persistente en XML

Arquitectura

El sistema sigue una arquitectura de microservicios con separación clara entre frontend y backend:

  • Programa 1 (Frontend): Aplicación web Django que consume la API
  • Servicio 2 (Backend): API REST Flask que procesa datos y gestiona almacenamiento XML
  • Comunicación: HTTP/JSON entre frontend y backend
